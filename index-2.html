Agent Reference
Signal Agent Commands

Everything you can do inside the XAU Signals Telegram agent. Open @XAUSignals and try them.

Member Commands
/start CODE
Activate your subscription using your invite code. Run this once when you first join. Example: /start APEX-X7K2
/mysub
Check your subscription status — shows your invite code, activation date, and exact expiry time so you always know how much access you have left.
/stats
View the agent's live performance stats — overall win rate, today's results, this week, this month, total trades fired, and average pips per trade.
/history
See the last 10 closed signals with results — pair, direction, grade, entry, exit, and outcome. WIN, LOSS, or EXPIRED with pip values.
/open
List all currently active open signals — pairs, direction, entry price, SL, TPs, and how long they've been live. Know what's in play at any moment.
/price PAIR
Get the current live price for any supported pair. Example: /price XAUUSD — returns the latest mid-price from Twelve Data.
/status
Check that the agent is online and running. Shows current scan schedule, session times, and next scheduled scan window.
Admin Commands — visible to subscribers for transparency
/signal Admin
Manually trigger a full scan across all session pairs right now — without waiting for the scheduled scan time. Useful for on-demand analysis.
/pair XAUUSD Admin
Run APEX analysis on a single specific pair immediately. Returns the full signal or a NO TRADE verdict with the reason.
/learning Admin
View what APEX has learned from its trade history — win rates by pair, grade, and session, plus what adjustments it's making to improve accuracy.
/shadow Admin
View the shadow trade report — signals that were filtered out and what would have happened if they executed. Shows whether filters are helping or hurting, and displays the current adaptive thresholds APEX is using.
/gencode LABEL Admin
Generate a new single-use invite code. The label helps identify who it was created for. Codes are valid for the subscription period from first use.
/members Admin
View all active subscribers — names, chat IDs, code used, activation date, and expiry. Full member roster at a glance.
/codes Admin
List all generated invite codes — used, unused, active, and revoked. Shows which codes have been activated and by whom.

All signals are delivered automatically at session opens — no commands needed.
Commands are for checking stats, managing your subscription, and on-demand analysis.
